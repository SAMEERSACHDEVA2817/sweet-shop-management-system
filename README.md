# 🍬 Lovely Sweets – Online Ordering & Management System

A complete web-based sweet shop management system developed using HTML, CSS, and JavaScript. The application allows customers to browse sweets, place orders, track orders, provide ratings, and interact with the shop online.

The system also provides an advanced admin dashboard for managing products, monitoring orders, tracking sales, and analyzing business performance.

---

## ✨ Features

### Customer Features

- User Registration and Login
- Product Categories
- Live Product Search
- Shopping Cart
- Order Placement
- Order Tracking
- Order History
- Notifications
- Customer Profile
- Ratings and Reviews
- Coupon System
- Photo Gallery
- Recently Viewed Products

### Admin Features

- Admin Login
- Sales Dashboard
- Revenue Analytics
- Order Management
- Update Order Status
- Product Availability Control
- Add New Products
- Customer Management
- Sales Charts and Reports
